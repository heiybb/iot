# Programming Internet of Things (1910)   
## Assignment 1: Python & Sense HAT programming  
The first assignment deals with Python and Sense HAT programming.

In this assignment you will create a Raspberry Pi application written in Python3 OOP code. You will be required to use certain modules of Raspberry Pi. The following learning objectives will be covered by this assignment:

CLO1: Understand IoT principles, design and abstraction of developing IoT systems
CLO3: Integrate a variety of IoT devices, sensors and services to build complex applications
CLO4: Learn the basics of Raspberry Pi and compatible programming frameworks
You will be using your Raspberry Pi Kit and Raspbian virtual machine to code the software using the in-built IDE for Python.
